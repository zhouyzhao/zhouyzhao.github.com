Python IDE环境搭建 eclipse+PyDev

十月 13, 2016

学习Python语言是因为它简洁，优雅的风格。刚开始学习Python时，我是直接使用Sublime text编辑器进行学习的。但一直学习的过程中，感觉脑子不好使，经常记不住函数的使用方法，而Sublime text的语法提示也没有详细的语法说明。权衡下，还是折腾IDE吧。

现在折腾的eclipse+PyDev方案，算是免费版中较好的一个方案。如果付费可以选择较为热门的pycharm，当然pycharm也有免费的社区版，但有部分功能被阉割了。

前期准备

  • JDK
  • python
  • eclipse
  • PyDev

（点链接，可跳转到相关应用的下载页）

安装eclipse之前需要安装JDK，而我安装的是JDK8和eclipse neon1版本。

所以先到java官网下载JDK然后进行环境配置：

先配置path变量

选择jdk8的安装目录：比如我安装在C盘下

输入变量值：;C:\Java\jdk1.8.0_05\bin


在配置classpath变量 需要配置tools.jar，dt.jar

输入变量值为:.;C:\Java\jdk1.8.0_05\lib\tools.jar;D:\Java\jdk1.8.0_05\lib\dt.jar

测试

点击‘开始-运行’

输入cmd 命令 打开DOS窗口

再输入javac

JDK配置好，就可以下载eclipse，eclipse直接下载，解压就可以使用。然后，我们就进行PyDev的下载配置。

打开eclipse→顶部工具栏（Help）→Install New Software

add→Name（PyDev）→输入下载地址：（http://pydev.org/updates） →OK

eclipse会查找到PyDev→勾选PyDev→取消勾选（Contact all update sites during install to find required software）→点Next 就可以开始安装了，在安装过程中，会弹出接受协议许可和证书验证，选择接受协议，点Finish，最后OK，稍等片刻，重启eclipse就安装完成。

配置PyDev→顶部栏（window）→Preferences→看到刚安装PyDev→点（Interpreters）→Python Interpreter→Auto Config就能自动配置电脑里的相关路径的python，python环境自动配置好，点OK就可以使用eclipse进行python编程了。
